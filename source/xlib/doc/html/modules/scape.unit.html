Module scape.unit

Prototype: unit and performance testing.

Offsprings of the unit prototype are considered unit tests. The functions of a unit test form a collection of independent tests which are executed in order of definition.

invoke (self)
Invoke unit. Setup and run all embedded tests. The instance table may contain the following options:
  • silent=true will suppress output
  • repetition=N will cause each test to run N times
  • name=Name set test unit name to Name

Run all tests. This method invokes reset() and setup() first, then runs all tests and finally calls the teardown() method. If self.silent is set it will run silently.
