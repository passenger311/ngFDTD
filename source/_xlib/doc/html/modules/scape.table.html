Create a clone of table structure by performing a deep copy. Metatables are shallow copied.

Parameters

Return value:

clone of data structure
fcopy (source, dest, filter)
Filter copy. The only entries which are copied from source to destination are those with keys in the filter table.

Parameters

  • source: source table
  • dest: destination table (optional)
  • filter: filter table (optional)

Return value:

dest

query (tab, strkey)
Query table by string key. Instead of tab["one"]["two"]["three"] for example one may use query(tab,"one.two.three").

Parameters

  • strkey: string key

Return value:

the referenced entry or nil if it does not exist.
valueset (array, value)
Create a set from array values. This function returns a table where the array values are the keys and the values are all set to the given value or true.

Parameters

  • array: array of keys
  • value: value [true]

Return value:

value set table
