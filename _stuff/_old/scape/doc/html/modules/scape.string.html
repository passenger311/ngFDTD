split (str, delim, plain, maxnum)
Split string. Partition string into a number of fragments using the specified delimiter. If the maximum number of fragments is set to nil then the string will be completely partitioned.

Parameters

  • str: string to split
  • delim: delimiter string or pattern [" "]
  • plain: use plain matching for delimiter [false]
  • maxnum: maximum number of partitions [str:len()]

Return value:

table of fragments
